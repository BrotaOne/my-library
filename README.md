## 其他
### 图书分类方法
简化版的《杜威十进制分类法》（DDC），用0-9 的大类编号

| 编号 | 类别         | 适用书籍示例                                   |
|------|--------------|--------------------------------------------|
| 000  | 计算机、信息、百科 | 计算机、人工智能、编程、百科全书             |
| 100  | 哲学与心理学   | 哲学、心理学、自我提升                       |
| 200  | 宗教         | 宗教、神学、灵修                             |
| 300  | 社会科学     | 政治、法律、经济、社会学                     |
| 400  | 语言         | 语言学、外语学习、写作                       |
| 500  | 自然科学     | 数学、物理、化学、生物、地球科学             |
| 600  | 应用科学     | 医学、健康、农业、工程、商业                 |
| 700  | 艺术         | 电影、摄影、音乐、绘画、设计                 |
| 800  | 文学         | 小说、诗歌、戏剧、散文                       |
| 900  | 历史与地理   | 中国史、世界史、地理、文化                   |
